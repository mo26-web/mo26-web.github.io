I am a PhD candidate in Chemical and Materials Engineering (Process Control) at the University of Alberta, supervised by Dr. Biao Huang. My research focuses on applying deep learning, probabilistic models, and reinforcement learning to real-world industrial challenges.

Prior to my PhD, I completed my M.Sc. at Khajeh Nasir Toosi University of Technology, graduating ranked 1st in my cohort with a GPA of 19.56/20. My thesis developed a recommender system via deep RL and GANs for gas turbine fault detection.

I am passionate about bridging the gap between AI research and industrial practice — from causal discovery in process systems to anomaly detection in networked environments.

Projects

Industrial Research Projects

2026
Supervisor
MPC Decision Support Toolbox — Interactive Web Application & AI Agent
Shell Canada · NSERC Alliance Program
Supervised the development of a full-stack Model Predictive Control decision support toolbox for industrial process monitoring and fault diagnosis. The system features KPI computation, anomaly detection, LSTM-based KPI trend forecasting, and transfer entropy causal analysis, delivered through a Next.js + FastAPI web application with both manual and AI-guided modes. The autonomous Expert Guidance Agent combines an LLM with local RAG over plant documents and a structured tool suite to conduct end-to-end fault investigations from a single natural language prompt. Guided the research direction, system architecture, and AI agent design throughout the project.
2025 – 2026
Lead Researcher
Performance Monitoring and Decision Support for MPC Using KPIs in Distillation Unit
Shell Canada · NSERC Alliance Program
Designed and implemented a data-driven MPC decision support system for control loop performance monitoring in an atmospheric and vacuum distillation unit. Developed a multi-category KPI evaluation framework — covering capacity, safety, control, and economic indicators — computed over sliding windows from process data. Applied LSTM-based multi-output forecasting for predictive KPI trend estimation and transfer entropy causal analysis for anomaly source identification. Validated against diverse anomaly scenarios including poor PID tuning, stealthy cyberattack (ramp attack), and human-induced tight constraints, with causality matrices confirming source detection alignment with simulation ground truth.
2025 – 2026
Supervisor
AI-Augmented Autonomous Oscillation Analysis Toolbox
Shell Canada · NSERC Alliance Program
Supervised the development of an LLM-integrated oscillation analysis system for industrial process control. The system evolved from an AI-assisted toolbox interface into a fully autonomous pipeline featuring Model Context Protocol (MCP) tool integration, plant-wide spectral envelope analysis with Oscillation Contribution Index ranking, live OPC UA server connectivity with scheduled streaming analysis, and a Retrieval-Augmented Generation (RAG) module with adaptive relevance filtering. Guided the research direction, technical architecture, and milestone delivery across two annual presentation cycles.
2025
Supervisor
Plantwide Oscillation Toolbox — Interactive Web Application
Shell Canada · NSERC Alliance Program
Supervised the development of an interactive web-based oscillation analysis toolbox built using Plotly Dash, providing a unified interface for single-loop and plant-wide oscillation detection, root cause analysis via stiction and poor tuning identification, and deep learning-based diagnostics including CNN with Markov Transition Field and GAF-SAE (Gramian Angular Field + Stacked Autoencoder) methods. Guided the research direction, algorithm selection, and UI/UX design decisions throughout the project.
2024
Lead Researcher
Oscillation Detection, Stiction Detection and Causal Analysis Toolbox
Shell Canada · NSERC Alliance Program
Designed and implemented an integrated GUI-based toolbox for automatic oscillation detection and root cause analysis in industrial control loops. Developed deep learning-based methods for oscillation detection (CNN with shape-based pattern recognition), stiction detection (Markov Transition Field + CNN), and poor controller tuning identification (Gramian Angular Field + Stacked Autoencoder). Introduced a physics-informed sparse Granger causality framework incorporating domain knowledge as optimization constraints with L1 regularization to eliminate spurious causations, producing interpretable causal maps for fault propagation analysis. Validated on a distillation column case study with 17 process variables.
